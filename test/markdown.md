Markdown
========

Most text entered into the Bedlam Admin site is rendered using Markdown (CommonMark/GFM).

---------------------------------------------------------

Paragraphs
----------

Leave a blank line between paragraphs.

This is one paragraph.

This is another paragraph.

End a line with two spaces to force a line break:  
Like this.

---------------------------------------------------------

Links
-----

Inline links:

[This](https://www.bedlamtheatre.co.uk) is Bedlam Theatre's website.

Relative links work too:

See the [About](/about/) page for details.

---------------------------------------------------------

Images
------

    ![Image description](/attachments/myimage)

---------------------------------------------------------

Headers
-------

# Heading 1

## Heading 2

### Heading 3

Setext style also works:

This is a major header
======================

This is a minor header
----------------------

---------------------------------------------------------

Emphasis
--------

*italic* and **bold** and ~~strikethrough~~

---------------------------------------------------------

Tables
------

| Column 1 | Column 2 | Column 3 |
|----------|----------|----------|
| Cell A   | Cell B   | Cell C   |
| Cell D   | Cell E   | Cell F   |

Aligned columns:

| Left | Centre | Right |
|:-----|:------:|------:|
| A    | B      | C     |

---------------------------------------------------------

Lists
-----

Unordered:

- Item one
- Item two
- Item three

Ordered:

1. First
2. Second
3. Third

---------------------------------------------------------

Code
----

Inline `code` like this.

Fenced code block:

```
def hello
  puts "Hello, world!"
end
```
